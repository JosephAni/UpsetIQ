# Screenshots

This directory contains screenshots of the UpsetIQ mobile application.

## Adding Screenshots

To add a new screenshot:

1. Save the image file in this directory with a descriptive name (e.g., `app-home-screen.png`)
2. Reference it in the README files using the path `docs/screenshots/filename.png`
